# 要件定義書: IPA単語帳アプリ

## 1. プロジェクト概要

IPA（情報処理技術者試験）の応用情報技術者試験に出現する用語を学習するための単語帳Webアプリケーション。スマートフォンでの利用を主目的とし、PWAとしてオフライン対応・ホーム画面追加が可能。

## 2. 対象ユーザー

- 応用情報技術者試験の受験者
- スマートフォンで隙間時間に学習したいユーザー

## 3. データソース

- ap-siken.com から取得した用語データ（5,488語）
- CSV形式: URL, 大分類番号, 中分類番号, 用語名, 読み, 英語, カテゴリ, サブカテゴリ, 説明

## 4. 機能要件

### 4.1 単語一覧表示

- カード形式で用語名を一覧表示する
- タップで説明文（意味）の表示/非表示を切り替えられる
- 各カードに正答率を表示する（記録がある場合）

### 4.2 ジャンル絞り込み

- 大分類（23カテゴリ）でフィルタリングできる
- 中分類（96カテゴリ）でフィルタリングできる
- 大分類 → 中分類のドリルダウン形式

### 4.3 検索

- 用語名・読み・英語でテキスト検索できる
- インクリメンタルサーチ（入力に応じてリアルタイム絞り込み）

### 4.4 出題モード

#### 4.4.1 出題設定

- ジャンル選択（大分類・中分類）
- 出題方向の選択
  - 用語名 → 意味
  - 意味 → 用語名
- 出題形式の選択
  - フラッシュカード型
  - 4択クイズ型
- 出題数の選択（10 / 20 / 50 / 全問）

#### 4.4.2 フラッシュカード型

- 問題（用語名 or 意味）を表示する
- タップで解答を表示する
- 「覚えた」「まだ」ボタンで自己申告する
- 結果を正答率として記録する

#### 4.4.3 4択クイズ型

- 問題（用語名 or 意味）を表示する
- 4つの選択肢を表示する（同じ中分類から優先的に出題）
- 選択後に正誤フィードバックを表示する
- 結果を正答率として記録する

### 4.5 進捗・正答率記録

- 用語ごとに正答数/出題数を記録する
- 大分類・中分類ごとの正答率サマリーを表示する
- 苦手用語リスト（正答率が低い順）を表示する
- 進捗データのリセット機能

### 4.6 PWA対応

- オフラインでの利用が可能
- スマートフォンのホーム画面に追加できる
- アプリライクな全画面表示（アドレスバーなし）

## 5. 非機能要件

### 5.1 対応環境

- iOS Safari / Android Chrome（スマートフォン）
- PC ブラウザ（補助的）

### 5.2 パフォーマンス

- 5,488語の一覧表示がスムーズに動作すること
- 検索・フィルタリングが体感0.5秒以内に反映されること

### 5.3 データ永続化

- localStorage を使用して端末にデータを保存する
- サーバーサイドのデータ保存は不要

### 5.4 ホスティング

- GitHub Pages で静的ホスティング
- 独自ドメイン不要

### 5.5 デザイン

- シンプル・ミニマルなデザイン（白背景）
- モバイルファースト

## 6. 将来の拡張（スコープ外）

- 間隔反復学習（忘却曲線ベースの出題調整）
- Capacitor / React Native によるネイティブアプリ化
- 複数端末間のデータ同期
